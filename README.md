# European-Football-Analytics-Dashboard

An interactive multi-league football analytics dashboard built with Python, Dash and Plotly.  
The application enables exploration of match statistics, team performance trends and FIFA player attributes.

---

## Features

### Team & Match Analytics
- Multi-league support:
  - Premier League
  - La Liga
  - Serie A
- Multi-season comparison
- Dynamic team selection
- KPI cards:
  - Matches played
  - Goals scored
  - Goal difference
  - Win rate
  - Shots conceded per game

### Visualizations
- Goals per team (home vs away)
- Goal difference trend over season
- Time series of team goals
- Discipline statistics:
  - Cards
  - Fouls
  - Corners
  - Shots

### Advanced Match Analysis
- Momentum chart (cumulative points)
- Best and worst matches radar comparison
- Match search and statistical comparison
- Betting odds visualization

### Player Analytics (FIFA dataset)
- Player search and comparison
- Radar charts:
  - Physical attributes
  - Defensive attributes
  - Attacking attributes
- Position-based analysis
- Rating vs potential heatmap
- Preferred foot and body type distributions
